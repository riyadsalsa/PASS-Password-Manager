# PASSi - Simple & Secure Password Manager

![iOS](https://img.shields.io/badge/iOS-15.0%2B-blue)
![Swift](https://img.shields.io/badge/Swift-5.9-orange)
![License](https://img.shields.io/badge/License-MIT-green)

A clean, private password manager that stores everything locally on your device. No cloud, no subscriptions, no third-party servers — just your passwords under your control.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 **Master Password** | Single password to unlock all your entries |
| 📱 **100% Local Storage** | Everything stays on your device — no cloud sync |
| 📤 **CSV Import/Export** | Easily move passwords between devices |
| 🔍 **Search** | Find any password instantly |
| 🎨 **Colored Passwords** | Visual password analysis (uppercase, lowercase, numbers, symbols) |
| 📋 **Easy Copy** | One-tap copy buttons for all fields |
| 🚫 **No Auto-Lock** | Stays open when switching apps — lock manually when done |

Open PASSi/PASSi.xcodeproj in Xcode
Select your team in Signing & Capabilities

🎯 How to Use

First Launch

Create a master password
Import passwords via CSV or add manually using the ➕ button
Daily Use

App stays unlocked when switching to other apps
Lock manually via Settings → Lock App Now
Copy any field by tapping the 📋 button or long-pressing text
CSV Format

csv
Name,Username,Password,URL,Notes
Gmail,user@gmail.com,password123,https://gmail.com,Primary email
