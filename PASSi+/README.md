## 🔒 Enhanced Security Features

### Built-in 2FA
- No Google Authenticator or external apps needed
- 6-digit code appears directly on the lock screen
- Toggle visibility with the 👁️ eye icon
- Session-based keys — new key generated every launch

### Recovery Codes
- 10 one-time use codes for emergency access
- Paper backup in case you lose your phone
- Generate new codes after recovery

### Crash Protection
- Journaling system prevents lockouts
- Automatic rollback if app crashes during sensitive operations
- Self-healing design
